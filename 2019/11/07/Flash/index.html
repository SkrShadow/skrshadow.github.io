手机刷机教程

跳到导航跳到搜索

现在很多安卓手机都有一个BL锁（Bootloader Lock），其主要目的是为了用户的隐私数据安全，以及防止线下经销商未经同意修改System Cust Modem Boot等分区，以及保护查找手机功能。但是对于想刷机的但是对于想刷机的童鞋来说，BL锁是必须要被解开的，当然并不是所有的机子都需要解BL锁，具体的自己去官网看看，或者询问客服，一般不需要解BL的是2014年8月以前的安卓手机。这类手机可以直接在Fastboot模式下使用命令行对System分区进行操作，比如刷机，之后的手机需要打开BL才能对其在Fastboot下对System等分区进行操作。

那么如何验证你有没有BL锁呢？

首先你需要一个ADB运行环境（Android Debug Bridge）

Code

下载并解压ADB.zip，将你解压到的文件分别复制入System32和SysWOW64下即可完成部署。

如需验证是否成功，可以管理员运行CMD，输入adb devices命令，如果成功回显则证明部署成功。

安装完成ADB运行环境之后，将手机进入Fastboot模式（一般为音量-和开机键同时3秒）
