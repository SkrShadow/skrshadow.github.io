Win7重装过程中出现提示框错误

有的时候在重装win7系统的时候，会出现一个提示框提示“Windows安装程序无法将Windows配置为在此计算机的硬件上运行”类似错误，常发于Intel第六代处理器之后，和AMD Ryzen系列及同代处理器之后。

就比如这样：

就这种错误

而且无论点 确定 还是点 关闭窗口都会重启

这个时候可以这样操作：

  1. 出现该提示框的时候不要点击关闭窗口，也不要点击确定

  2. 同时按下 shift键 和 F10键呼出命令提示符窗口 (如果没反应，考虑该电脑是否需要再同时按下FN键才能使用F10键)

  3. 在命令提示符中输入命令

    Code

    注意空格

    ，然后按下回车键执行该命令(切换目录)

  4. 再输入

    Code

    然后按下回车键执行该命令启动msoobe程序直接进入win7初始化界面

  5. 完成后如果还回到错误弹窗界面的话，选择确定，然后重启系统就应该正常了。
