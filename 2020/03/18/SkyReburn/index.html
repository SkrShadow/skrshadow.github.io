项目说明

这个包就是第一个版本的封装，我都没想到过去多少年在这个200GB西数蓝盘里面备份的这个包有一天还能重见天日，还能被我重制233333

版本号

Ver 1.1.1 Final Based on Windows XP

底包版本

Windows XP Professional SP3

预置应用

QQ影音（已更新至最新）

Bandzip最后一版支持XP的

Office2010

SkyPE 2.0 的重制版本

实用工具组，放置在桌面，用完可以删

更新说明

版本：1.0-1.1.1 Final

（上次更新时期（文件修改日期）2013.8月份）

补丁日期：2012.06-停止更新前最后一版

移除大量以前花里胡哨的东西，比如各种壁纸主题（我当时为什么脑抽要往里面放这些）

移除系统休眠，虚拟内存，系统还原点产生的文件（系统大小从12GB直接跳水到5GB可还行…）

封装工具使用了ES3最后一版更新。也算是个时代的结束吧。

万能驱动底包更新到支持XP的最后一版

移除各种奇奇怪怪的文件，进行系统的轻度精简。

封装后打包工具Ghost 11.2（大概）——WinImage

依旧是配合SkyPE4.0食用风味更佳（小声bb）

另外，这个版本大概也是XP系列的最后一版了，收拾NAS老硬盘意外发现的我这个XP包，于是打算重制一下。这个版本也算是为了纪念一台从小陪到我长大的电脑吧。双核奔腾，1GB的DDR2内存，9600GT显卡，这个系统就是在那个电脑变成我的第一台NAS之前的XP系统制作的。内部文件各种清理算是尽力抹掉我以前的使用痕迹了。可能有些被忽略掉了，就请无视吧。还折腾了两遍WindowsXP安装盘修复那时候被我乱删乱改的系统文件2333，和以前的自己斗智斗勇就很绝。不愧是我.jpg

这个系统的安装方式直接使用CGI就可以了。或者其他能读取ESD文件的安装器。

为了防止文件损坏用7-zip又压了一遍，别忘了解压。
